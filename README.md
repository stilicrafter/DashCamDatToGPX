
# DashCamDatToGPX
This repo contains files, that allow you to use the .DAT file from Vantrue DashCams as input and output .GPX Files.

German Version see below

## How to use

Use Python to launch main.py.
Depending on your configuration: Click twice on the main.py or open PowerShell and type  py main.py while you are in the main folder. 
This is the GUI Application where you can select a source and destination file.
Select the .DAT file from the VanTrue and destination for the .GPX File.

You can also use pyFiles as a CLI  and pass it source and destination. See comment in the File.
